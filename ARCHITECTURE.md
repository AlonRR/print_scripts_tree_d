# Architecture & Guiding Principles

## Core pattern: library → script → output

```
shapes/            main.py           models/
(pure functions) → (compose+export) → (STL/3MF files)
```

Geometry logic and file I/O are always separate.

---

## Folder layout

```
print_scripts_tree_d/         ← repo root
│
├── print_scripts_tree_d/     ← the importable Python package
│   ├── shapes/
│   │   ├── __init__.py       ← re-exports the full public API
│   │   ├── primitives.py     ← make_washer and basic solids
│   │   ├── panels.py         ← make_hexagonal_mesh, lattice, sheet parts
│   │   └── furniture.py      ← make_table, make_column, assembled objects
│   ├── params.py             ← dataclasses for named parameter sets
│   └── export.py             ← save_stl(): export + trimesh validation
│
├── tests/
│   └── test_shapes.py        ← geometry unit tests (volume, bounds, errors)
│
├── models/                   ← generated STL/3MF output (git-ignored)
├── main.py                   ← build script: load params → compose → export
└── sandbox.ipynb             ← interactive exploration + live 3D preview
```

---

## Layer responsibilities

### `shapes/` — geometry library
- Pure functions only. Never touches the filesystem.
- Each function takes numbers/primitives, returns a `Compound`.
- Split by domain: primitives / panels / furniture.
- `shapes/__init__.py` re-exports everything — callers import from `shapes`, not submodules.

### `params.py` — configuration
- Dataclasses (`HexPanelParams`, `ColumnParams`, `TableParams`) hold all tuneable values with defaults.
- Different configurations are version-controlled as data, not scattered magic numbers.
- `main.py` constructs a params object; shape functions receive individual values from it.

### `export.py` — export + validation
- `save_stl(shape, path)` wraps `export_stl` and immediately validates the mesh with trimesh.
- Asserts watertight and positive volume before returning — fails fast if the model is not print-ready.
- All file I/O goes through here; shape functions stay side-effect-free.

### `main.py` — build script
- Load params → call shape functions → call `save_stl`.
- No geometry logic lives here; no raw numbers except via params.

### `tests/` — geometry contracts
- Test measurable invariants: volume, bounding box size, error conditions.
- Do not test visual appearance.

### `sandbox.ipynb` — interactive scratchpad
- Explore and iterate with `show(shape)` (ocp-vscode live viewer).
- Not a build step; nothing here is required to produce output.

### `models/` — generated output
- Contains only exported STL/3MF files. Never hand-edited.
- Git-ignored binary artifacts — regenerated by running `main.py`.

## Adding a new shape domain

1. Create `print_scripts_tree_d/shapes/<domain>.py` with pure functions.
2. Re-export from `shapes/__init__.py`.
3. Add a `<Domain>Params` dataclass to `params.py` if it has tuneable parameters.
4. Add corresponding tests to `tests/test_shapes.py`.

---

## Export validation

`export.py`'s `save_stl()` runs automatically after every export:

```python
mesh = trimesh.load(str(path))
assert mesh.is_watertight   # no holes — required for slicing
assert mesh.volume > 0      # normals point outward
```

Validation is on by default and can be skipped with `validate=False` for intermediate
exports during development.

## Tool roles

| Tool | Role |
|---|---|
| **build123d** | BREP solid modeling — precise, parametric, print-ready geometry |
| **trimesh** | Mesh validation — watertightness, volume, normals |
| **numpy** | Math for point arrays and transforms |
| **ocp-vscode** | Live 3D viewer in VS Code via `show(shape)` |
| **ruff** | Linter + formatter (replaces flake8 + black + isort) |
| **mypy** (strict) | Static type checking |
| **pytest** | Geometry unit tests |
| **hatchling** | Build backend — makes the folder an installable package |

---

## Output formats

- **STL** — universal, accepted by all slicers.
- **3MF** — preferred when color, materials, or units metadata matter.
- Prefer STL for single-material mechanical parts; 3MF for everything else.
